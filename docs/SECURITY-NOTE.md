This tool is a portable Windows utility created as part of MPTech Windows Tools.

Official sources:

- https://github.com/xml2811/SSL-Checker
- https://mptechsolutions.es

Important:

- Do not download this tool from unofficial websites.
- Early builds may not be digitally signed.
- Windows SmartScreen may show a warning.

If you are unsure, download only from the official GitHub Releases page.

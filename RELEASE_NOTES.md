# MPTech SSL Checker v1.0.0

First public release of MPTech SSL Checker.

## Highlights

- Portable Windows executable.
- HTTPS/TLS certificate validation.
- SSL health score.
- Hostname coverage check.
- SAN and wildcard SAN detection.
- SHA256 fingerprint and serial number.
- Signature algorithm and certificate size.
- DNS/TCP/TLS/certificate error classification.
- TXT export with save dialog.
- English, Spanish and Portuguese interface.

## Download

Use the portable executable:

- MPTech-SSL-Checker-v1.0.0-portable.exe

The .sha256 file can be used to verify the download integrity.

## Note

Windows SmartScreen may show a warning because this release is not code-signed yet.
